Standard Assignment Naming Format for Pakistani Universities

Most Pakistani universities require assignment files to follow a specific naming convention before submission. The typical format is:

RollNumber_CourseCode_TaskName.ext

For example: FA22-BSCS-0001_OOP_Assignment3.pdf

Use the Prefix field for your roll number, Subject for the course code, Task for the assignment number, and Underscore as the separator. The tool handles multiple files — if you upload 3 files, they will be named with sequential numbers at the end.

How to Rename Files for FAST, NUST, COMSATS and Other Universities

  • FAST-NUCES: Format: RollNo_CourseName_Task.pdf — e.g. i221234_DS_Lab5.pdf
  • NUST: Format often includes student name and registration: Name_RegNo_Subject_Task.pdf
  • COMSATS: Typically: SP22-BSCS-001_CS201_Assignment2.pdf
  • UET: Department-specific — check your course outline for the required format

Set the fields to match your university's required format and check the live preview before downloading.
