How to use RGBA colors in CSS

RGBA is the most versatile color format in CSS. Use the rgba() function in any color-accepting property:

  • background-color: rgba(0, 242, 254, 0.5); — semi-transparent background
  • border: 1px solid rgba(255, 255, 255, 0.15); — frosted glass border
  • box-shadow: 0 4px 20px rgba(0, 0, 0, 0.3); — soft drop shadow
  • color: rgba(148, 163, 184, 0.8); — slightly dimmed text

The fourth value (alpha) ranges from 0 (fully transparent) to 1 (fully opaque). Adjust the A slider in this tool to set your target transparency, then copy the RGBA card.

RGBA vs RGB vs HEX — which format should you use in CSS?

HEX (#RRGGBB) Best for solid colors. Shortest syntax. Used in Figma, Sketch, and most design tools.
RGB rgb(r,g,b) Explicit channel control. No transparency. Ideal when you need exact numeric values.
RGBA rgba(r,g,b,a) Best for transparency. Overlays, shadows, glassmorphism. Industry standard for layered UI.
HSL hsl(h,s%,l%) Best for theming — easy to adjust brightness and saturation. CSS custom property systems.
CSS Variable --c: #hex Best for design systems. Define once, use across all components via var(--color).
Tailwind bg-[#hex] Arbitrary values for one-off brand colors outside your Tailwind config palette.

What is the difference between RGBA and ARGB?

RGBA and ARGB contain the same four color channels — red, green, blue, and alpha — but in different byte order. RGBA is used in CSS and web development, where the alpha comes last: rgba(255, 99, 71, 0.5). ARGB is used in Android development and some design APIs like Figma's plugin system, where alpha comes first as a hex byte: #80FF6347 (where 80 is the hex value of 50% opacity).

This tool outputs RGBA format for CSS. To get an ARGB value for Android, take the alpha percentage from the A slider, convert it to a hex byte (0–FF), and prepend it to the HEX output.

RGBA Color Picker for WCAG Accessibility Contrast

The contrast preview panel below the output cards shows how white and black text appear on your selected RGBA color in real time. This is essential for meeting WCAG 2.1 contrast ratio requirements — a minimum of 4.5:1 for normal text and 3:1 for large text. Before deploying any color to a button, heading, or body text, check the preview to confirm readability for all users including those with visual impairments.

Frequently Asked Questions

What is an RGBA color picker?
An RGBA color picker lets you select a color visually or by entering a HEX code, then outputs it as an rgba(r, g, b, a) value. The alpha channel (a) controls transparency from 0 (invisible) to 1 (solid). This tool also outputs HEX, RGB, HSL, HSLA, CSS variable, SCSS variable, and Tailwind arbitrary class simultaneously.
What is the difference between RGBA and ARGB?
Both formats use the same four channels (red, green, blue, alpha) but in different order. RGBA is the CSS standard — alpha is last: rgba(255, 99, 71, 0.5). ARGB is used in Android and some design APIs — alpha byte is first: #80FF6347. This tool outputs RGBA for CSS use.
How do I convert HEX to RGBA?
Paste your HEX code (e.g. #FF6347) into the input field below the color preview. The RGBA card updates instantly. Use the A slider to set transparency. The default alpha is 1 (fully opaque). Click the RGBA card to copy.
How do I use RGBA in CSS?
Use rgba() in any CSS property that accepts a color value — background-color, color, border-color, box-shadow, outline, and more. Example: background-color: rgba(0, 0, 0, 0.5); creates a 50% transparent black overlay. RGBA is supported in all modern browsers.
What is the alpha channel in RGBA?
The alpha channel is the fourth value (a) in rgba(r, g, b, a). It controls opacity: 0 = fully transparent, 1 = fully opaque. rgba(0, 0, 0, 0.5) is 50% transparent black — commonly used for modal overlays. Use the A slider in this tool to set any alpha value and see the output update live.
How do I change the opacity / alpha value?
Drag the A (Alpha) slider — the bottommost slider in the RGBA controls. Moving it left decreases opacity toward 0 (transparent); right increases toward 100 (opaque). The RGBA and HSLA output cards update instantly.
What is the difference between RGB and RGBA?
RGB has three channels (red, green, blue) and is always fully opaque. RGBA adds a fourth alpha channel for transparency control. rgb(255, 99, 71) and rgba(255, 99, 71, 1) look identical — the difference only matters when alpha is below 1.
Does this color picker work with Tailwind CSS?
Yes. The Tailwind Arbitrary card outputs a class like bg-[#FF6347] you can paste directly into any Tailwind project. For RGBA with transparency in Tailwind v3+, define a CSS custom property via the CSS Variable card and reference it in your tailwind.config.js as a custom color token.
