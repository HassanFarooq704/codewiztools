Free QR Code Generator — URL, Text, WhatsApp and WiFi

This QR code generator creates standard QR codes for five common use cases: website URLs, plain text, email addresses, WhatsApp messages, and WiFi network credentials. The QR code generates in real time as you type — adjust the size slider and click Download PNG to save it for use in print or digital materials. All generation runs in your browser using QRCode.js — no data is sent to any server.

How to Create a QR Code for WhatsApp

Select WhatsApp type, enter the phone number with country code (e.g. +923001234567 for Pakistan, +447911123456 for UK) and an optional pre-written message. The QR code encodes a wa.me deep link. When someone scans it, WhatsApp opens with that number and message ready to send — no need to save the contact first.

How to Create a QR Code for WiFi

Select WiFi type and enter your network name (SSID), password, and security type. The QR code encodes the standard WiFi format: WIFI:T:WPA;S:YourNetwork;P:password;;. Android 10+ and iOS 11+ can scan this to connect automatically without typing the password — useful for guest networks, offices, and cafes.

What Size Should My QR Code Be?

  • Digital screens and websites: 200–300px — sharp enough for any display
  • Business cards: 300–400px — readable from 20–30cm scan distance
  • Posters and flyers: 400–512px — readable from 50cm+ scan distance
  • Banners and signage: Use 512px and scale up in your design software
