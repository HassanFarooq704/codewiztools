Free PDF Merger — Combine PDFs for Assignment Submissions

This PDF merger combines multiple PDF files into one document using PDF-Lib, a modern browser-based PDF library. No images are re-rendered — pages are copied directly from each source file, so the merged PDF preserves the original text quality, vector graphics, and formatting exactly.

How to Merge PDF Files for University Assignment Submission

Many universities require students to submit a single PDF file combining a cover page, main report, appendices, and references. Here is how to use this merger:

  • Upload all your PDF files — cover page, report sections, appendices
  • Use the ↑ ↓ arrows to arrange files in the correct order
  • Click Merge PDFs to combine them into one file
  • Download the merged PDF and submit to your university portal

Merge PDFs Without Losing Quality

Unlike PDF compressors, this merger does not re-render or compress any pages. It copies page data directly between PDF documents using PDF-Lib. The merged output has the same text sharpness, image quality, and vector graphics as the original files — no quality loss of any kind.
