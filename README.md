# CodeWizTools - 100% Private Client-Side Developer Utilities 🚀

CodeWizTools is a powerful, lightweight, and fully browser-based suite of web utilities designed for developers, designers, and content creators. 

Unlike traditional online tools, **CodeWizTools processes 100% of your data locally on your device.** Your data never touches a server, ensuring absolute privacy and security.

🔗 **Live Link:** https://codewiztools.com/

---

## 🛠️ Features & Tools Included

- **Color Picker & RGBA Converter:** Generate HEX, RGB, and RGBA codes with a built-in interactive palette.
- **Lorem Ipsum Generator:** Generate clean, placeholder dummy text with custom paragraph lengths for your UI layouts.
- **URL Encoder & Decoder:** Securely encode or decode strings for web transmission.
- **Base64 Converter:** Fast and private binary-to-text encoding right inside your browser.
- **JSON Formatter:** Validate and beautify complex JSON structures instantly.

---

## 🏗️ Architecture & Tech Stack

This project is built using a pure, highly-optimized client-side architecture to guarantee maximum speed and zero server overhead:

- **Frontend:** Semantic HTML5, Responsive CSS3 (Flexbox/Grid), and Modern Vanilla JavaScript (ES6+).
- **Core Engine:** Client-side DOM manipulation. No external trackers or invasive scripts.
- **SEO Ready:** Fully compliant with schema markup (`WebApplication` JSON-LD) and optimized for search intent.

---

## 🔒 Privacy First

Because all tools run natively via client-side JavaScript, this application requires:
- ❌ NO database logs
- ❌ NO server-side processing
- ❌ NO user data collection
